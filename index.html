Open Source FPS for the Web

Plasma Forks is a 3D first-person shooter with a sci-fi theme. Armed with a powerful Forkgun you take on an army of killer robots.

System Requirements

  • Latest Chrome (better) or Firefox
  • A good GPU
  • Decent CPU and enough RAM to handle JavaScript

Features

  • Action packed single-player campaign
  • Editor for creating and sharing more levels
  • High quality textures, models etc
  • Rigid body dynamics, i.e. physics
  • Dynamic lighting with shadows
  • Post-processing effects
  • Spatial audio

Open Source

This game is open source under the very permissive MIT license. Assets have different but still free licenses.
